1.
    git commit --amend: dùng để thay đổi commit message mới nhất bằng lệnh VIM.
    git commit --amend-m”<message>”: dùng để thay đổi commit message gần nhất bằng cách nhập commit message mới.
    git restore --staged <file>:  dùng để đưa từ vùng Staging về Working directory.
    git reset HEAD~1:  Dùng lệnh "git reset HEAD~1" (undo 1 commit mới nhất, tăng số lượng sau dấu "~" nếu muốn undo nhiều commit hơn nữa theo thứ tự từ trên xuống dưới).
